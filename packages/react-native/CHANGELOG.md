# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-14

### Added
- Initial release of @healthsync/react-native
- Health Connect integration for React Native
- Native module for Android Health Connect API
- React hooks for health data access
- TypeScript support with full type definitions
- Automatic permission handling
- Real-time data sync capabilities
- Background sync support
- Comprehensive test coverage

### Features
- **Health Connect Integration**: Full support for Android Health Connect API
- **React Hooks**: Easy-to-use React hooks for health data operations
- **Type Safety**: Complete TypeScript definitions
- **Permission Management**: Automatic permission request handling
- **Real-time Sync**: Subscribe to health data changes
- **Background Sync**: Sync health data in the background
- **Data Types**: Support for all major health data types (steps, heart rate, sleep, etc.)
- **Caching**: Efficient local caching with SQLite
- **Conflict Resolution**: Automatic conflict detection and resolution

### Platform Support
- Android: API 26+ (Android 8.0+)
- iOS: Coming soon

### Documentation
- Complete API documentation
- React Native integration guide
- Health Connect setup instructions
- Example applications

### Dependencies
- @healthsync/core: ^1.0.0
- React: >=16.8.0
- React Native: >=0.60.0
